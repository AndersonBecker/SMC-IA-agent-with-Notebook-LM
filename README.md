
# Sistema de Medição Centralizada(SMC)

Aqui vemos um projeto inicial onde foi criada um agente de IA para auxiliar pessoas a entender melhor o motivo da medição fiscal estar sendo implantada no Brasil.

## Desenvolvimento do projeto

este projeto será desenvolvido com o passar dos dias, pois há uma boa dificuldade de conseguir documentos técnicos para melhor compreensão do funcionamento do SMC.

** A princípio, o principal motivo do SMC é fazer com que os medidores fiquem isolados em uma caixa blindada no topo do poste, dificultando a possibilidade de fraudes e com isso evitando perdas financeiras das concessionárias de energia e deixando de encarecer a conta de luz de pessoas honestas que pagam corretamente sua conta.
